# Pizza_sales_dashboard
The Pizza Sales Dashboard provides a comprehensive analysis of sales performance across different pizza categories, sizes, and time periods. The dataset includes details such as order date, pizza type, category, size, quantity sold, and revenue generated. The objective of the dashboard is to monitor sales trends, identify top-performing products, and evaluate overall business performance

This dashboard was build using the following tools & technologies:
Tableau Desktop - Main visualization platform used for report creation
Excel- Data transformation and cleaning layers for reshaping and preparing the data

Key Features:
●  KPI cards displaying Total Revenue, Total Orders, Total Pizzas Sold, and Average Order Value.
● Sales trend analysis by date and time to identify peak business hours.
● Category-wise and size-wise revenue breakdown.
● Top 5 and Bottom 5 pizzas based on revenue and quantity sold.
● Interactive filters for date range, category, and size for dynamic analysis
